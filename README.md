Інструкція з відтворення стартової сторінки

Цей документ містить опис структури та ключових кроків для створення аналогічної сторінки "Лабораторні роботи 8 клас".

1. Технологічний стек

Для коректної роботи сторінки використовуються:

HTML5 — базова розмітка.

Tailwind CSS — для стилізації без написання окремих CSS-файлів.

Font Awesome — бібліотека іконок.

JavaScript — для автоматичної генерації 10 однакових карток.

2. Структура документа (Head)

У секції <head> обов'язково підключаються зовнішні скрипти:

<script src="[https://cdn.tailwindcss.com](https://cdn.tailwindcss.com)"></script>
<link rel="stylesheet" href="[https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css)">


3. Секція заголовка (Header)

Використовуйте контейнер з білим фоном (bg-white) та нижньою межею (border-b).

Для розміщення тексту та фото поруч використовуйте Flexbox (flex flex-col md:flex-row).

Фото зошита: Використовуйте клас notebook-img з налаштуванням max-width: 280px та заокругленням кутів.

4. Логіка генерації карток (Main Grid)

Замість того, щоб копіювати код для кожної з 10 робіт, використовується цикл for у JavaScript:

Масив назв: Створіть масив const labs = [...], де по порядку впишіть назви всіх робіт.

Цикл: Запустіть цикл for (let i = 1; i <= 10; i++).

Генерація HTML: Всередині циклу використовуйте document.write або маніпуляції з DOM для вставки блоку картики.

Шляхи посилань:

Для тестів: lab${i}/test.html

Для завдань: lab${i}/task.html

5. Стилізація карток

Кожна картка повинна мати:

Білий фон (bg-white), рамку (border-slate-200) та легку тінь (shadow-sm).

Ефект при наведенні (hover:translate-y-[-3px] та transition).

Кнопки з чіткими підписами: "Тест" та "Творче завдання".

6. Адаптивність

Для сітки карток використовуйте класи Tailwind:
grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-2
Це забезпечить відображення в одну колонку на телефонах і у дві на планшетах/ПК.
